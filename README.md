## 🚀 개발 환경 세팅 (Development Setup)

> 💡 처음 개발 환경을 세팅하는 팀원은 **아래 순서를 그대로 따라 하면 됩니다.**

### 1️⃣ Node.js 설치
1. [Node.js 공식 사이트](https://nodejs.org/) 접속  
2. “**LTS (안정 버전)**” 다운로드  
3. 설치 후 터미널(또는 Git Bash)에서 아래 명령으로 버전 확인:
   ```bash
   node -v
   npm -v
   
2️⃣ 프로젝트 클론하기

터미널(Git Bash) 열고 아래 명령 입력 👇

git clone https://github.com/사용자명/2025-2-2-web-project.git
cd 2025-2-2-web-project

3️⃣ 개발 브랜치로 이동

현재 모든 개발은 dev 브랜치에서 진행됩니다.

git checkout dev

4️⃣ 패키지 설치

package.json에 정의된 라이브러리를 자동으로 설치합니다.

npm install

5️⃣ 로컬 서버 실행

서버를 실행하려면:

node app.js


브라우저에서 http://localhost:3000
 접속
